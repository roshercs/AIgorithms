Clasificación Logística:

Las regresiones son funciones matemáticas que permiten inferir un modelos dada una colección de datos. En otras palabras, la función nos permitirá estimar o aproximar los datos dados, tanto los existentes como los nuevos. Esta característica es ideal para el aprendizaje automático, pues nuestro modelo se adecuará a los datos (aprende de ellos).

Aprendizaje automático

Aprender consiste en adquirir conocimientos (comprensión) mediante el análisis o la experiencia. Las personas aprendemos diariamente en base de lo que vivimos, lo que nos rodea, y los datos existentes de la situación. Nos adecuamos a lo que tenemos y actuamos en correspondencia a ello.

El aprendizaje automático consiste en aprender automáticamente a partir de los datos, a través de un proceso de inferencia, modelo o aprendizaje de ejemplos (reconocimiento de objetos o personas). La extracción automática de información es útil mediante la construcción de buenos modelos inferenciales. Con base a estos modelos inferenciales podemos realizar el pronóstico y la clasificación.

Pronóstico: Modela funciones de valor continuo, es decir, predice valores numéricos desconocidos. Clasificación: Predice etiquetas de una o más clases de tipo discretas (0, 1, 2) o nominales (A, B, C; o positivo, negativo; y otros).

En ambos casos, pronóstico y clasificación, si la precisión es aceptable, se utiliza el modelo para pronosticar o clasificar nuevos datos, cuyos valores o etiquetas no se conocen. Existen así pues, diferentes algoritmos para esto:

  • Regresión (lineal o logística)
  • Nearest Neighbor (kNN)
  • Arboles de Decisión
  • Máquina de soporte de vectores
  • Redes Neuronales Artificiales
  • Métodos Bayesianos
  • Entre otros

Regresión lineal

La regresión lineal es una forma básica de aprendizaje supervisado, cuyo objetivo es calcular una ecuación que minimiza la distancia entre la línea ajustada (recta) y todos los puntos de datos. La regresión según el número de variables son: si se tiene dos variables se trata de un problema de regresión lineal simple; si se tiene más de dos variables se trata de un problema de regresión lineal múltiple

Regresión Logística

La regresión logística es otro tipo de algoritmo de aprendizaje supervisado cuyo objetivo es predecir valores binarios (0 o 1). Este algoritmo consiste en una transformación a la regresión lineal. La transformación se debe a que una regresión lineal no funciona para predecir una variable binaria. La forma de el aprendizaje automático requiere tener una muestra para entrenar el modelo y otro para validarlo (no la misma).
Para este algoritmo se utiliza la misma estructura que la regresión lineal, pero se transforma la variable respuesta (0 o 1) en una probabilidad. Para esta transformación se utiliza la función logística (conocida también como sigmoide).

Validación de la clasificación

Para la validación del modelo se utiliza la matriz de clasificación. En la variable clase (la variable a clasificar u obtener) el conjunto de entrenamiento toma dos valores posibles: 0 o 1; positivo o negativo; falso o verdadero. Los valores positivos y negativos que se predicen correctamente se conocen como verdaderos positivos (VP) y verdaderos negativos (VN), respectivamente. Mientras que los valores clasificados se denominan falsos positivos (FP) y falsos negativos (FN).

Criterio de división: Para este método de evaluación del modelo se necesita dividir los datos en Datos de entrenamiento (training) y de los Datos de Prueba(test). El procedimiento es el siguiente: una vez generado el modelo (a partir de los datos de entrenamiento), se evalúa el modelo con los datos de entrenamiento. Se evalúan todos los elementos y se determina si la predicción (clase) coincide con los valores reales (Y). Se cuentan todos los elementos y se muestran los totales obtenidos en la matriz.
Con la matriz generada (ilustrada en la figura de arriba) podemos obtener los siguientes factores:

  1. Exactitud (Acurracy): Es el porcentaje de datos clasificados correctamente.
  2. Precisión: Es el porcentaje de clasificación positiva.
  3. Tasa de error (Misclassification Rate): Porcentaje de datos clasificados incorrectamente.
  4. Sensibilidad (True Positive Rate): Es el porcentaje de clasificación del total positivos.
  5. Especificidad (True Negative Rate): Es el porcentaje de clasificación del total negativos.

Estos valores nos permiten evaluar al modelo generado. De acuerdo a estos es que estimamos la fiabilidad del modelo y su utilidad real.

Así pues, esta matriz puede utilizarse de forma más específica para evaluar el modelo. La curba ROC por ejemplo.El concepto de ROC y AUC se basa en el conocimiento de la matriz de confusión, sensibilidad y especificidad. Este concepto puede aplicarse a cualquier algoritmo de clasificación. La curva ROC es una medida de rendimiento para problemas de clasificación. ROC (características operativas del receptor) es una curva de probabilidad y AUC (área bajo la curva) representa el grado de separabilidad. Cuanto mayor sea el AUC, mejor será el modelo para clasificar (distinguir) entre nuevos elementos. La curva ROC se genera trazando la tasa de los verdaderos positivos -TPR- (eje y) contra la tasa de los falsos positivos -FPR- (eje x).
